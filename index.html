Preparing and Mastering Data in PowerBI

In this Project, we prepare and master LendingClub data in Microsoft PowerBI. Before preparation, we understand data by generating descriptive statistics. Then, moving to data cleaning. We correct or remove dirty data; or treating it as missing. If outliers are detected, we treat them as missing if caused by input error; otherwise, we winsorize or seperate & study seperately. For missing data, we either delete, relace, or impute it. The last step in this project is transformation. This step is used to minimize bias in model predictions. We choose a transformation approach based on whether the input variable is Numeric or Categorical Rounding, Scaling and Non-linear transformation are the techniques to transform numeric variables to Numeric. When transfering Numeric Variables to Categorical, we use discretization or binning.

TikTok User and Revenue Forecast in Excel

In this project, I forecasted the user and revenue growth for TikTok. Firstly, I built a framework to infer about the total market size for apps, social media apps usage and demographics data. Then, I determine M, the total market size for TikTok mobile app. Once I made my case for M, I plugged in the 2017-2023 data numbers to the Bass model. Then, I Found the best combination of p (innovation) and q (imitation) parameters that creates a reasonably good fit.Based on the M, p, q parameters, I projected TikTok user growth and total TikTok users globally for the next 5 years. Lastly, I analyzed the company's renuve source and tested how Tiktok's revenue growth corresponds to E-Commerce industry growth.
